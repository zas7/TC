The Transfer-Client is a Java application for transfering data via Sedex. It offers a graphical user interface to the end-user for controlling the sending process. In detail, the Transfer-Client provides the following tasks:
  • Task 1: Waiting for incoming result files.
  • Task 2: Moving the result files to the stage folder.
  • Task 3: Creating sedex envelope files.
  • Task 4: Moving the result files together with the generated envelope files to the outbox sedex folder.
  • Task 5: Waiting for receipts.
